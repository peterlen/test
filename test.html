Used to load one or more AOI geometries from a file
 
File Format:   Each geometry is a list of comma-separated coordinates
Coordinate Format:   Decimal Degrees (longitude,latitude)
Geometry Delimiter:   Each Geometry is listed on a new line
Buffer Usage:   Values used only if Point or Line geometries exist
 
Example:   -76.239,22.45,-79.8754,23.7659
